
# Titanic Data Ingestion & Cleaning Pipeline

This project demonstrates a data ingestion workflow using pandas in Jupyter Notebook.  
It covers loading raw data, handling missing values, normalizing column names, performing basic exploratory data analysis (EDA), and exporting a cleaned dataset.

## Features Implemented

- CSV data ingestion using pandas  
- Missing value handling (median, mode, column removal)  
- Basic EDA using `head()`, `describe()`, `value_counts()`  
- Export of cleaned dataset
